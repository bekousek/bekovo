---
name: Experiment vs Activity classification
description: How to classify content into experiments (pokusy) vs activities (aktivity) on bekovo.cz
type: feedback
---

Pokus (experiment) = something that demonstrates/verifies a physical phenomenon or hypothesis. Either:
- Kvalitativní: observe something happening (light bends, magnet attracts)
- S měřením: measure values, get numbers (measure resistance, speed, density)

Aktivita = hands-on work that isn't strictly proving/observing a phenomenon:
- Making things (výroba periskopu, kalimby, elektromotoru) → activity, NOT experiment
- Games (soutěže, hry, kvízy)
- Teaching methods (peer instruction, formativní přístup)
- Group work (skupinové projekty)
- Could even be homework (domácí úkol)

**Why:** User corrected that "Výroba periskopu" was misclassified as experiment. Making something is an activity. An experiment tests a hypothesis or verifies a physical observation.

**How to apply:** When classifying content from sborníky, ask: "Does this TEST or OBSERVE a physical phenomenon?" If yes → experiment. "Does this involve MAKING, PLAYING, or DOING something hands-on?" If yes → activity.
